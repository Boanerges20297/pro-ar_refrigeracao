Atenção Usuários do Gmail

O Google exige o uso de uma **Senha de App** para sistemas externos. Sua senha comum do e-mail não funcionará.

Clique aqui para criar sua Senha de App no Google

Atenção Usuários do Yahoo Mail

O Yahoo exige o uso de uma **Senha de App** gerada no painel de segurança.

Clique aqui para criar sua Senha de App no Yahoo

Atenção Usuários do Outlook / Hotmail

Para contas com Verificação em Duas Etapas, você deve usar uma **Senha de App**.

Clique aqui para gerenciar suas Senhas de App na Microsoft

Atenção Usuários do iCloud

O iCloud exige a criação de uma **Senha Específica de App** no seu ID Apple.

Clique aqui para criar sua Senha de App na Apple
